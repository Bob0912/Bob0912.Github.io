一起共同冲刺~ 一.return简单使用下面来一个Java程序，表示的是在** self1这个包 中我们创建了一个名叫 demo2的Class类 ** ：

package self1;
public class demo2 {    public static void main(String[] args) {        for(int i=1;i<=10;i++) {            if(i==5) {                return;//表示方法的结束后续语句将不会执行            }            System.out.println(i);        }    }}

程序输出：

在输出中我们可以看到并没有打印5之后的数字，这是为什么呢？

这里就要提到我们所要说的return，表面上意思是返回，在程序中我们可以想象成某个数据或者字符。在实际场景中呢，也确实有这个作用，但这也是作用之一。

接下来我们来看看 return到底有什么具体用法：

  1. 返回方法指定类型的值(这个类型是确定的)，也可以是对象。
  2. 方法的结束可以有两种形式，第一种是有返回类型的，例如：return i; 第二种是无返回类型比如直接return;

To：方法前有void代表无返回值，没有void表示有返回值！

**二.Java基础程序小结
由于之前小编也是停更了一段时间，所以这里先演示一个简单的Java程序作为小结啦~ 代码块: **

package self1;
public class demo3 {    public static void main(String[] args) {        for(int i=1;i<=9;i++) {            for(int j=1;j<=i;j++) {                System.out.print(j+"*"+i+"="+(j*i)+" ");            }            System.out.println();        }    }}

程序输出：

至于为什么要 **使用乘法表作为基础语法的小结呢？使用程序编写Java代码只是简单入门，但至少可以 Show your Code！ 所以乘法表包含了我们之前写的如何使用IDEA编写程序，如何写一个主函数以及运行，更多的我们还学习了各种数据类型以及运算符的使用，我们可以在包中新建我们所需要的类，然后再类中完成我们所想要完成的功能。 给自己写一个小小的功能也算得上一天中最快乐的事之一，嘿嘿
