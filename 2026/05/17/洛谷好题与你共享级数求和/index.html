题目描述

**
**

输入格式

一个正整数 k.

输出格式

一个正整数 n.

输入输出样例

输入 复制

1

输出 复制

2

说明/提示

【数据范围】

对于100%的数据，1<=k<=15

** 源代码 如下： **

#include <stdio.h>int main() {    int k,n;    double sum;    sum=0.0;    k<=1&&k>=15;    scanf("%d",&k);    n=1;    for(;sum<=k*(1.0);n++)    {        sum += (1.0 / n);    }    n--;    printf("%d",n);    return 0;}
