这里说一下g++在编译过程中的步骤：

  1. 预处理，生成.i的文件

  2. 将预处理后的文件转换成汇编语言，生成文件.s

  3. 由汇编变为 目标代码 （机器代码）生成.o的文件

  4. 连接目标代码，生成可执行程序
