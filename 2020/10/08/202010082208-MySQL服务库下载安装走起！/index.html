那么什么是 MySQL数据库 呢？

简单来说，MySQL是一个多用户、多线程且开源的的SQL数据库，是一个客户机/服务器结构的应用，它由一个服务器守护程序mysqld和很多不同的客户程序和库组成。

至于为什么选择MySQL作为我的数据库呢，当然主要原因是因为它的 快 咯，目前市场上运行最快的 SQL (Structured Query
Language结构化查询语言) 数据库之一就是MySQL Server，由瑞典的T.c.X.
DataKonsultAB公司开发。而且它还提供了其它数据库少有的编程工具，MySQL对于个人用户也是免费的，当然，开发app还是需要支付一定费用的，详见官网。（https://www.mysql.com/）

然后找到并点击path,再点击新建，在其后面添加: 你的mysql bin文件夹的路径（如小编的是）：D:\Program
Files\mysql-8.0.21-winx64\bin

由于我们的mysql文件中没有默认的配置文件,所以需要我们新建一个.ini的配置文件,新建文件名字为 my-default.ini
（新建一个文本文件，然后修改文件后缀名）里边内容如下:

接下来就可以通过命令行来运行MySQL啦，切记一定要通过管理员身份打开，否则后面会报错权限不够！！！

命令如下: mysqld -install 如果出现以下提示,说明安装成功。

开始启动服务命令为: net start mysql 这时候会出现报错，如下：

系统提示无法启动,并报错为3534 ,我们通过 mysqld –console
命令来查看错误日志，系统提示我们在MySQL里面木有找到\data文件

这时需要我们新建一个\data文件，我们用命令进入到mysql文件夹的路径,输入以下命令: mysqld –initialize-insecure
–user=mysql

关于命令行怎么进入MySQL文件夹的路径，我们需要先键入cd\，进入盘符选择，然后打D:即可进入MySQL的目录，只是要一直在刚刚的命令行中，若是重新打开就得自己cd打开MySQL的文件目录，详细命令过程如下：

执行完上面命令后，MySQL会自建一个data文件夹，并且建好默认数据库，登录的用户名为root，密码为空。到这里MySQL服务就启动成功啦！

服务启动成功之后，我们可以输入命令： mysql -u root -p （第一次登录没有密码，直接按回车即可），会出现如下窗口：
