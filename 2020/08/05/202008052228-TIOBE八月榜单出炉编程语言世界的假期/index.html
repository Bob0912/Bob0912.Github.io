八月刚到，TIOBE已经为我们准备好了最新的编程语言排行榜，相信很多程序员都会时不时关注一下排行榜的变化,虽然仅起参考作用,但无疑也是个饭后吐槽点

TIOBE社区能坚持每个月都推送最新的数据是蛮厉害的,并且收集了很多高级工程师,课程以及第三方供应商提供的数量,比如有:Google, Bing,
Yahoo!, Wikipedia, Amazon, YouTube和百度等.

在本月中，榜单变化相较于上个月其实变化不大。 C语言 依然领头羊 ，且保持着强劲的增长势头。且与 第二名 Java 之间差距逐月增大，从上个月相差 1.35% 的份额逐步增长到 2.55% 的差额， Python 依然排在第三位 。

在上个月中， R语言 被称为“ 史无前例的高光时刻 “,且本月继续保持上升的势头，
按照TIOBE的说法，R语言有望成为2020年TIOBE的年度编程语言。 在数据分析和统计计算等领域，R语言成为开发者最受欢迎的编程语言之一。回看
2019 年 8 月，R 语言还处于第 20 位，一年后，一连上升了 12 个名次，到达了第 8
位，所以R语言是很有可能是2020年TIOBE的年度编程语言。
