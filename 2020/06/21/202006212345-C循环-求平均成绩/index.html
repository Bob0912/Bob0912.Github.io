任务描述

本关任务：编写一个程序，输入学生人数和每个人的成绩，计算平均成绩。注意：当输入的学生人数小于等于0时，输出平均成绩为0分！

编程要求

编程实现：编写一个程序，输入学生人数和每个人的成绩，计算平均成绩。

以下是测试样例：

输入：3 90 70 80

输出：the number of students:the scores:average=80.00

源代码：

#include<stdio.h>int main(){    int n,i;    double sum=0,temp;    printf("the number of students:");    scanf("%d",&n);    if(n<0){        printf("the scores:average=%.2f",sum);    }else{        for(i=0;i<n;i++){            scanf("%lf",&temp);            sum+=temp;        }        printf("the scores:average=%.2f",sum/n);    }    return 0;}
