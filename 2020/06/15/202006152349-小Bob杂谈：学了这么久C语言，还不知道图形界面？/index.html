**Don’t cry because it is over, smile because it happened.

不要因为结束而哭泣，微笑吧，为你的曾经拥有。

最近因为小编忙着C语言课程设计去啦，所以停更了两天，对于每天等着小编发推文的小伙伴要说声不好意思啦，不得不说，写课设确实会学到很多知识，当然，前提是你必须自己写啦



。

分享一首最近在听的歌啦：

今天就不和大家谈论代码啦，聊聊小编在写课设时的感想，虽然还没写完，哈，主要是得自己找各种头文件已经功能函数，老师讲的实际上是很少的，所以得告诫下届的学弟学妹们学东西得自己去找，不要全靠老师来教，老师教的东西是很少很少的，比如C语言的各种头文件函数，老师教的可能不会超过十种，而实际上头文件是有非常多的，估计资深程序员也很难说清楚有多少种，这么多头文件也只能分常用和不常用，所以更多的只能靠自己去寻找啦。

小编的课设是关于图书管理系统，不造大家有木有写过哈。

小编试了后发现如下代码（主要部分）能写出图形界面：

 MessageBox(            NULL,            TEXT("你好，欢迎来到此图书管理系统"),            TEXT("现在，可以开始你的操作了"),            MB_OKCANCEL | MB_ICONINFORMATION    );

上面的代码需声明在如下头文件中：

#include <windows.h>

当然，还有一些与图形界面相关的函数，这里小编也不过多谈及了，毕竟小编也才最近接触，以后学完了再和大家分享咯，现在想要自己学会的，还得靠大家自己去寻找啦。
