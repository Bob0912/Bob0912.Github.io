本关任务：

编写一个程序，任意输入n，求S=1!+2!+…+n!。注意：n!表示n的阶乘。0的阶乘等于1，负数的阶乘等于0。提示：(n+1)!=n!*(n+1)

例如：

输入：10 输出：4037913

输入：7 输出：5913

输入：-1 输出：0

源代码：

#include<stdio.h>int main(){    int n,i,c=1,s=0;    scanf("%d",&n);    for(i=1;i<=n;i++)    {        c*=i;        s+=c;    }    if(n == 0){        printf("1");    }else if(n<0){        printf("0");    }else{        printf("%d",s);    }    return 0;}
