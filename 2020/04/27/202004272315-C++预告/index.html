小编准备开始学习C++啦，从C++的绪论来看，C++和C差别其实不是特别大，不过由于C++是支持面向对象的，所以还是会有一定的不同。

所以学完C的小伙伴其实也可以入手C++，相信学起来也不难。小编还是觉得两门语言是可以同时学起来的，特别是C和C++还有很多共同点。

关于C小编也会更新的啦。

下面是小编入手C++写的一个简单程序：

#include <iostream>//C++的头文件 using namespace std;int main() {    cout << "Hello" ;    cout << " World!" <<endl;    cout <<"Welcome to C++!";    return 0;}
