本关任务：

从键盘输入两个学生的学号，姓名和成绩（整数），分别存入结构体中，输出成绩较高的学生的学号，姓名和成绩。

测试输入：

  1. 14308100238 xiaowang 99

  2. 15408122584 xiaoli 78

预期输出：

14308100238 xiaowang 99

源代码：

#include<stdio.h>struct stu{ long long int id; char name[20]; int grade;}stu1,stu2;int main(){ int n; scanf(“%lld %s %d”,&stu1.id, stu1.name, &stu1.grade); scanf(“%lld %s %d”,&stu2.id, stu2.name, &stu2.grade); if(stu1.grade<stu2.grade){ printf(“%lld %s %d”,stu2.id, stu2.name, stu2.grade); } else{ printf(“%lld %s %d”,stu1.id, stu1.name, stu1.grade); } return 0;}
