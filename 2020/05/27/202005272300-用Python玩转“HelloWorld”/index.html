Hello World的条件输出

描述

获得用户输入的一个整数，参考该整数值，打印输出”Hello World”，

要求：

如果输入值是0，直接输出”Hello World”

如果输入值大于0，以两个字符一行方式输出”Hello World”（空格也是字符）

如果输入值小于0，以垂直方式输出”Hello World”

源代码：

n = eval(input())#需掌握eval的用法if n == 0:#判断    print("Hello World")elif n > 0:    print("He\nll\no \nWo\nrl\nd")else:    for c in "Hello World":#循环        print(c)

当然这个写法有很多，比如你也可以不用for循环，直接在每个字符后面加个换行符。

小编在开始输入数字时是用的如下这种（也可使用，转换为int类型）：

n=input("")#input输入的算是字符串，所以下面需要转换一下n=int(n)
